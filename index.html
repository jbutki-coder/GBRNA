About This Project

A.S.I.S and the Fellowship Service Conference

Grey Book Reflection is a literature project that is still in the draft stage, however many groups use it as NA Literature.

A.S.I.S. is directly responsible to the NA Groups it serves. This site is maintained to support daily reflection, literature study, review/input, and carrying the NA message.

The uploaded project file is titled Gray Book Reflections. The public site uses the project title Grey Book Reflection.

In Loving Service: Grey Area Historical Literature Study Group of Narcotics Anonymous
