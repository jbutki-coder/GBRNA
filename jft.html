Just For Tonight

Just For Tonight is another A.S.I.S. daily reading draft. The source file describes it as an evening meditation reading.

This page uses the same daily reading setup as Grey Book Reflection: today’s reading, previous/next, archive, search, copy link, and review/input.

The source project file is titled Just for Tonight. The public site uses the title Just For Tonight.
